# Experiments Directory

This directory is grouped by artifact type:

- `raw_demos/`: raw Isaac demonstration rollouts.
- `lerobot_datasets/`: converted LeRobot datasets.
- `training_runs/`: policy training runs, checkpoints, and summaries.
- `eval_videos/`: policy evaluation runs, saved frames, and mp4 files.
- `camera_debug_runs/`: camera debug captures and camera-specific reports.
- `smoke_runs/`: smoke-test output directories.
- `reports/`: standalone text reports and logs.
- `docs/`: experiment plans and design notes.
